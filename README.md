# MediSupply — Medical Instruments Supplier Website

  A premium, single-page React website for a medical instruments & surgical equipment supplier.

  ## Requirements

  - Node.js 18 or higher (https://nodejs.org)
  - npm (comes with Node.js)

  ## Quick Start

  ```bash
  npm install
  npm run dev
  ```

  Then open http://localhost:5173 in your browser.

  ## Build for Production

  ```bash
  npm run build
  ```

  The production files will be output to the `dist/` folder. You can serve them with any static host.

  ## Preview Production Build

  ```bash
  npm run preview
  ```

  ## Tech Stack

  - React 19 + TypeScript
  - Vite 7
  - Tailwind CSS 4
  - Framer Motion (animations)
  - shadcn/ui components
  - react-hook-form + Zod (contact form validation)
  - lucide-react (icons)
